# 바이브 코딩 템플릿
`claude code`를 사용하여 바이브 코딩을 시작하기 위한 프로젝트 템플릿입니다.

`claude code`는 `plan mode`로 시작해 주세요.

# Process
해당 템플릿으로 바이브 코딩을 진행하면, 다음 프로세스가 수행됩니다.

1. 유저가 요구사항 명세 입력
2. 유저와 에이전트가 대화하며 요구사항을 명확히 정리
3. 에이전트가 `./spec` 경로에 제품 스펙 문서 작성 (`SDD(Spec-Driven Development)`)
4. 에이전트가 테스트 코드 작성 (`TDD(Test-Driven Development)`)
5. 에이전트가 구현 진행
6. 에이전트가 lint 및 테스트 수행 (이슈가 발견될 경우, 스스로 해결)
7. 에이전트가 `ai-review` 스킬을 사용하여 코드 리뷰 수행 (이슈가 발견될 경우, 스스로 해결)

# Install
## 기본 설정 (필수)
`README.md`를 삭제하고, `CLAUDE.md`를 적절히 변경해 주세요.
- 폴더 구조의 변경이 필요할 경우, 원하는 구성을 작성하여야 합니다.
- 추가적인 프로세스나 참고할 내용들을 추가로 작성해 주세요.

## 스택 구성 (옵션)
특정 언어나 프레임워크를 희망하는 경우, 수동으로 설치하는 것이 안전합니다.

만약, 폴더의 구조가 변경된다면 `CLAUDE.md`를 적절히 변경해 주세요.
